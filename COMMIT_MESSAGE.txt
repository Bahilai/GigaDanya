## Breaking Changes:
- API endpoint changed from Foundation Models to Agent API
- Request/response structure completely different
- Requires YANDEX_AGENT_ID in configuration

## Migration Notes:
Users need to:
1. Create an agent in Yandex AI Studio
2. Add YANDEX_AGENT_ID to local.properties
3. Rebuild project to generate new BuildConfig

## Benefits:
- Flexible agent configuration without code changes
- Managed prompts and instructions via AI Studio UI
- Support for tools, knowledge bases, and integrations
- Agent versioning and A/B testing
- Better separation of concerns

## Technical Details:
- Base URL: https://rest-assistant.api.cloud.yandex.net/v1/
- Endpoint: POST /responses
- Request: { prompt: { id, variables }, input, stream }
- Response: { output: [{ content: [{ text, type }] }], status, error }
